# 观点科技电子招投标交易平台 — 数据库设计

> 依据《观点招投标web版功能清单v0603-1》及《研发需求文档 V1.4》更新。技术栈：MySQL 8.0 / InnoDB / utf8mb4。
> 功能对照：[03-功能清单v0603](./03-功能清单v0603.md)
> 命名规范：表名小写下划线，主键 `id`（BIGINT），业务表带 `create_by/create_time/update_by/update_time/deleted`。
> 完整可执行 DDL 见 [../sql/schema.sql](../sql/schema.sql)。

---

## 0. 通用约定

- 审计字段：`id` / `create_by` / `create_time` / `update_by` / `update_time` / `deleted`。
- 金额 `DECIMAL(15,2)`；文件统一走 `attachment` + 对象存储；外键采用**逻辑外键**（建索引、应用层校验）。
- 多角色：`sys_user_role` 多对多；激活角色在 JWT 中维护。

## 1. 用户、角色与组织

| 表 | 说明 | 关键字段/枚举 |
|---|---|---|
| `sys_user` | 用户 | username, password, phone, wx_openid, status |
| `sys_role` | 角色 | code: GUEST/BIDDER/TENDERER/MANAGER/EXPERT/SALES/ADMIN |
| `sys_user_role` | 用户-角色(多对多) | audit_status(专家/经理角色需审核) |
| `supplier_profile` | 投标人企业资料 | company_name, credit_code, 收款银行/账户 |
| `expert_profile` | 专家资料 | major(专业), title(职称), sign_img_id(电子签名) |
| `agency_company` | 代理公司 | name, credit_code（项目经理"我的代理公司"） |
| `role_application` | 角色申请 | apply_role: MANAGER/EXPERT；audit_status |
| `tenderer_invite` | 邀请招标人 | inviter_id(经理), invitee_phone/name, status:待接收/接收/拒绝 |
| `user_preference` | 我的偏好 | industries/regions/types(用于商机推荐) |
| `cms_banner` | 首页广告轮播 | title, image_id, link_url, sort, status |
| `cms_site_link` | 常用网站链接 | name, url, sort, status |

```sql
CREATE TABLE sys_role (
  id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  code VARCHAR(32) NOT NULL COMMENT 'GUEST/BIDDER/TENDERER/MANAGER/EXPERT/SALES/ADMIN',
  name VARCHAR(32) NOT NULL,
  UNIQUE KEY uk_code(code)
) COMMENT='角色';

CREATE TABLE role_application (
  id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  apply_role VARCHAR(32) NOT NULL COMMENT 'MANAGER/EXPERT',
  major VARCHAR(128) NULL COMMENT '专家专业',
  attach_id BIGINT NULL COMMENT '证明材料',
  audit_status TINYINT DEFAULT 0 COMMENT '0待审 1通过 2驳回',
  audit_remark VARCHAR(255), audit_by BIGINT, audit_time DATETIME,
  -- + 审计字段
  KEY idx_user(user_id), KEY idx_audit(audit_status)
) COMMENT='角色申请(项目经理/专家)';

CREATE TABLE tenderer_invite (
  id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  inviter_id BIGINT NOT NULL COMMENT '邀请人(项目经理)',
  invitee_user_id BIGINT NULL, invitee_name VARCHAR(64), invitee_phone VARCHAR(20),
  status TINYINT DEFAULT 0 COMMENT '0待接收 1已接收 2已拒绝',
  -- + 审计字段
  KEY idx_inviter(inviter_id)
) COMMENT='邀请招标人';
```

## 2. 招标项目

### tender_project 招标项目（核心 / 双维状态）
```sql
CREATE TABLE tender_project (
  id              BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  project_no      VARCHAR(40)  NOT NULL COMMENT '项目编号',
  name            VARCHAR(200) NOT NULL,
  section         VARCHAR(100) NULL COMMENT '标段',
  procurement_type VARCHAR(20) NOT NULL COMMENT 'PUBLIC/INVITE/INQUIRY/SINGLE/NEGOTIATION/CONSULTATION',
  tender_type     VARCHAR(16)  NULL COMMENT '招标类型:工程ENGINEER/货物GOODS/服务SERVICE',
  industry        VARCHAR(32)  NULL COMMENT '行业类型',
  region          VARCHAR(64)  NULL,
  budget          DECIMAL(15,2) NULL,
  manager_id      BIGINT NOT NULL COMMENT '项目经理user_id',
  tenderer_id     BIGINT NULL COMMENT '招标人user_id',
  agency_id       BIGINT NULL COMMENT '代理公司id',
  file_fee        DECIMAL(15,2) NULL COMMENT '文件费',
  platform_fee    DECIMAL(15,2) NULL COMMENT '平台使用费',
  status          VARCHAR(20) NOT NULL DEFAULT 'DRAFT'
                  COMMENT 'DRAFT/BIDDING/OPENING/OPENED/AWARDED/FINISHED/ARCHIVED/ABORTED',
  eval_node       VARCHAR(20) NOT NULL DEFAULT 'NOT_STARTED'
                  COMMENT 'NOT_STARTED/REVIEWING/NEGOTIATING/SECOND_QUOTE/FINISHED',
  reg_start       DATETIME NULL, reg_end DATETIME NULL,
  bid_open_time   DATETIME NULL COMMENT '开标时间',
  eval_total_score DECIMAL(8,2) NULL COMMENT '评审总分',
  price_score_method TINYINT NULL COMMENT '报价评分方式(1/2/3)',
  archived        TINYINT DEFAULT 0 COMMENT '是否归档',
  content         MEDIUMTEXT NULL COMMENT '招标公告正文',
  bid_file_id     BIGINT NULL COMMENT '招标文件附件(解析带入字段)',
  -- + 审计字段
  UNIQUE KEY uk_no(project_no), KEY idx_status(status), KEY idx_manager(manager_id)
) COMMENT='招标项目';
```

## 3. 报名与缴费

### bid_registration 投标报名（细化状态）
```sql
CREATE TABLE bid_registration (
  id            BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  project_id    BIGINT NOT NULL,
  supplier_id   BIGINT NOT NULL COMMENT '投标人user_id',
  company_name  VARCHAR(128) COMMENT '报名公司',
  contact_name  VARCHAR(64),  contact_phone VARCHAR(20),
  apply_file_id BIGINT COMMENT '报名文件',
  audit_status  TINYINT DEFAULT 0 COMMENT '0待审核 1通过 2驳回',
  audit_remark  VARCHAR(255) COMMENT '驳回原因',
  reg_status    VARCHAR(20) DEFAULT 'PENDING'
                COMMENT 'PENDING待审核/UNPAID待缴费/SUCCESS报名成功/REJECTED已驳回/CANCELLED已取消',
  bid_status    VARCHAR(20) DEFAULT 'NONE'
                COMMENT 'NONE/BIDDING投标中/OPENED已开标/WON中标/LOST未中标/INVALID废标',
  reg_time      DATETIME COMMENT '报名时间',
  -- + 审计字段
  UNIQUE KEY uk_project_supplier(project_id, supplier_id)
) COMMENT='投标报名';
```

### bid_document 投标文件（加解密）
```sql
CREATE TABLE bid_document (
  id              BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  registration_id BIGINT NOT NULL,
  attach_id       BIGINT NOT NULL COMMENT '投标文件附件',
  bid_price       DECIMAL(15,2) NULL COMMENT '投标报价',
  duration        VARCHAR(64) NULL COMMENT '工期',
  encrypt_pwd     VARCHAR(128) NULL COMMENT '六位解密密码(加密存储)',
  encrypted       TINYINT DEFAULT 0 COMMENT '是否已加密',
  decrypt_status  TINYINT DEFAULT 0 COMMENT '0未解密 1已解密',
  decrypt_time    DATETIME NULL,
  sign_img_id     BIGINT NULL COMMENT '签字解密的签名图',
  submit_time     DATETIME NULL, withdraw_time DATETIME NULL,
  -- + 审计字段
  KEY idx_reg(registration_id)
) COMMENT='投标文件(加密/撤回/解密)';
```

### payment_order 缴费订单 + payment_order_item 明细
```sql
CREATE TABLE payment_order (
  id              BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  order_no        VARCHAR(40) NOT NULL,
  project_id      BIGINT NOT NULL,
  registration_id BIGINT NULL,
  payer_id        BIGINT NOT NULL COMMENT '付款人',
  biz_type        VARCHAR(20) NOT NULL COMMENT 'REGISTER报名缴费/AGENCY代理费',
  total_amount    DECIMAL(15,2) NOT NULL,
  pay_channel     VARCHAR(16) NULL COMMENT 'WECHAT/ALIPAY/TRANSFER对公汇款',
  status          TINYINT DEFAULT 0 COMMENT '0未支付 1已支付 2已退款',
  pay_time        DATETIME NULL,
  fee_mode        TINYINT NULL COMMENT '代理费:1固定计费 2标准计费(折扣)',
  discount        DECIMAL(5,2) NULL COMMENT '标准计费折扣',
  -- + 审计字段
  UNIQUE KEY uk_order(order_no), KEY idx_reg(registration_id)
) COMMENT='缴费订单(报名缴费/代理费)';

CREATE TABLE payment_order_item (
  id       BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  order_id BIGINT NOT NULL,
  fee_type VARCHAR(16) NOT NULL COMMENT 'FILE文件费/PLATFORM平台使用费/AGENCY代理费',
  amount   DECIMAL(15,2) NOT NULL,
  KEY idx_order(order_id)
) COMMENT='缴费订单明细';
```

```sql
  respond_time DATETIME NULL, sign_time DATETIME NULL,
  -- + 审计字段
```

### evaluation_item 评审项 + evaluation_score 评分
```sql
CREATE TABLE evaluation_item (
  id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  project_id BIGINT NOT NULL,
  type VARCHAR(20) NOT NULL
       COMMENT 'FORMAL形式/QUALIFY资格/RESPONSE响应性/COMMERCE商务/TECH技术/PRICE报价',
  name VARCHAR(128), max_score DECIMAL(6,2), sub_total DECIMAL(6,2) COMMENT '该类总分',
  sort INT,
  -- + 审计字段
  KEY idx_project(project_id)
) COMMENT='评审项配置(六类评审)';

CREATE TABLE evaluation_score (
  id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  project_id BIGINT NOT NULL,
  registration_id BIGINT NOT NULL COMMENT '被评投标人',
  expert_id BIGINT NOT NULL,
  item_id BIGINT NOT NULL,
  pass TINYINT NULL COMMENT '形式/资格/响应性:0否1是',
  score DECIMAL(6,2) NULL COMMENT '商务/技术/报价评分',
  remark VARCHAR(255), submitted TINYINT DEFAULT 0 COMMENT '0暂存 1已提交',
  -- + 审计字段
  UNIQUE KEY uk_score(registration_id, expert_id, item_id)
) COMMENT='评分(专家×投标人×评审项)';
```

### negotiation 谈判磋商 / second_round_quote 二轮报价（竞谈/竞磋）
```sql

CREATE TABLE second_round_quote (
```

## 5. 通用与内容

| 表 | 说明 | 关键枚举 |
|---|---|---|
| `attachment` | 附件元数据 | biz_type: bid_doc/apply/license/article/report... |
| `notification` | 消息通知 | type: AUDIT/INVITE/OPEN/NEGOTIATION/AWARD... |
| `complaint` | 投诉建议 | category(意见/投诉) + sub_type；status(待处理/处理中/已回复)；reply/handler/handle_time |
| `article_category` | 内容栏目 | guide指南/policy法规/case案例解析/download下载/news信息动态 |
| `article` | 文章/文档 | category_id, attach_id |
| `search_hot_keyword` | 热门搜索统计 | keyword, search_count |
| `search_history` | 用户历史搜索 | user_id, keyword, search_time |
| `guarantee` | 保函(待补充) | user_id, project_id, amount, status |
| `invoice` | 发票申请(待补充/业务结算) | user_id, amount, status |
| `operation_log` | 操作审计日志 | module, action, biz_id, detail |

```sql
CREATE TABLE complaint (
  id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT, project_id BIGINT NULL,
  category VARCHAR(16) COMMENT 'ADVICE意见/COMPLAINT投诉',
  sub_type VARCHAR(32) COMMENT '平台优化建议/平台报错问题/招标信息质疑/招标文件质疑/投标投诉质疑/中标投诉质疑',
  title VARCHAR(200), content TEXT, attach_id BIGINT,
  status TINYINT DEFAULT 0 COMMENT '0待处理 1处理中 2已回复',
  reply TEXT, handler BIGINT, handle_time DATETIME,
  -- + 审计字段
  KEY idx_user(user_id)
) COMMENT='投诉建议';

CREATE TABLE search_hot_keyword (
  id BIGINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
  keyword VARCHAR(64) NOT NULL,
  search_count BIGINT DEFAULT 0,
  UNIQUE KEY uk_keyword(keyword)
) COMMENT='热门搜索统计';
```

## 6. ER 概要

```text
sys_user 1—N sys_user_role N—1 sys_role
sys_user 1—1 supplier_profile / expert_profile ; 1—N role_application ; 1—N tenderer_invite
agency_company 1—N tender_project
tender_project 1—N announcement / evaluation_item / expert_assignment / award
tender_project 1—N bid_registration 1—N bid_document
bid_registration 1—N payment_order 1—N payment_order_item
tender_project 1—N negotiation / second_round_quote (N—1 bid_registration)
evaluation_score: registration × expert × item (三维唯一)
tender_project 1—1 evaluation_report 1—N report_doc
```
