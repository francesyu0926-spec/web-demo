## 6. v0603 与旧版差异（实现须遵守）

| 规则 | v0603-1 要求 | 实现要点 |
|---|---|---|
| 技术评分 | 所有专家均可评分 | `TECH` 项：各专家独立打分，汇总取平均 |
| 商务/报价评分 | **仅专家组长** | `COMMERCE`/`PRICE` 项：仅 `is_leader=1` 的专家可提交；汇总取组长分数 |
| 推选组长 | 专家互选组长 | 签到后、评分前，`POST .../elect-leader` |
| 谈判/二轮报价 | 组长发起 | 后续接口须校验 `is_leader=1` |
| 招标人 | 首页功能可见 | 招标人专属「我的项目」 | `GET /api/tenderer/projects`（邀请接受后绑定项目） |
